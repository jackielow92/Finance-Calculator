# Legacy Files (v1)

These files are from the original single-file implementation of Finance Calculator.

## Files

- **index.html** - Original single-file application (147KB) containing all HTML, CSS, and JavaScript
- **pcb_table.js** - Malaysian PCB tax lookup table data (2.1MB)
- **pcb_script_tag.txt** - Script tag reference for pcb_table.js

## Migration

These files were migrated to a modern React + TypeScript architecture in v2.0.0.

The new implementation is located in `/packages/frontend/`.

## Note

These files are kept for reference only. The active codebase is in the `packages/` directory.
